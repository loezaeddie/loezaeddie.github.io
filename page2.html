My favorite project I have ever done in my years of being in App Academy has to be the Weather Animals Project. The weather animals project was where we can put a city, state, or country and it would show the temperature of that area and a cute picture of cute bulldogs. In this project we use a free weather API from APIXU. The API allows us to find the temperature of an area. The link to the API is right HERE.
